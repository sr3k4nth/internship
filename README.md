# Exchange Rates Manager App

### Requirements

- [Node 16.16.0](https://nodejs.org)
- NPM 6.x.x+

## Start Exchange Rates Manager server

`npm start`

## Start Data Ingestion service

`npm run db:ingestion <file_name.csv>`

Example: `npm run db.ingestion 2022.csv`
